What is a web browser?

A web browser is a software application used to access informtion on the World Wide Web.

Examples include;

  • Mozilla Firefox.
  • Microsoft Edge.
  • Internet Explorer.

Functions of the web browser

  1. Provide a way for users to access an navigate web pages.
  2. Display the web pages properly.
  3. Provide technology to enable multimedia features.
  4. Provide access to internet services(such as FTP and e-mail).
  5. Perfrom authentication and encryption features.

What are internet protocols?

These are methods or protocols by which data is sent from one computer to another on the internet.

Types of internet protocols

TCP/IP(Transmission Control Protocol/Internet Protocol)

SMTP(Simple Mail Transfer Protocol)
This allows a device to send and deployto email to a remote endpoint with a TPC connection.
HTTP(HyperText Transfer Protocol)
This enables systems to communicate on the World Wide Web.
